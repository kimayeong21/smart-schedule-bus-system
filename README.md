# 🚌 스마트 일정 및 버스 통합 관리 시스템

## 📌 프로젝트 개요
개인 일정 관리와 실시간 버스 정보를 하나의 시스템으로 통합하여  
효율적인 시간 관리를 지원하는 프로그램입니다.

## 🚀 주요 기능
- 회원가입 / 로그인
- 일정 등록 / 수정 / 삭제
- 버스 도착 정보 조회
- 관리자 기능 (사용자 및 데이터 관리)

## 🛠 개발 환경
- 개발 언어: Python (Flask)
- 데이터베이스: MySQL

## 📁 프로젝트 구조
